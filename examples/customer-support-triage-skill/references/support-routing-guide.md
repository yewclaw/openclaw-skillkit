# Support Routing Guide

## Urgent
- Active production outage affecting multiple customers.
- Payment processing failure that blocks checkout or renewal.
- Security or account-access issue that could expose customer data.

## High
- Core workflow is blocked for a single customer or workspace.
- Data import/export is failing without a safe workaround.

## Routing Hints
- Billing and invoices -> finance-ops
- Authentication and account access -> identity-platform
- Broken integrations -> integrations
- UI defects without data loss -> product-support

## Customer Reply Pattern
- Confirm the issue was received.
- State the team reviewing it.
- Ask only for the next missing detail that unblocks investigation.
